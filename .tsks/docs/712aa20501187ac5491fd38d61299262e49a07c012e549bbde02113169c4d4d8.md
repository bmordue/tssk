Phase 3: Create Homebrew formula and other package manager distributions

## Phase 3 – v1.0

**Goal:** Production-ready v1.0 release with a stable public API, comprehensive documentation, and distribution packaging.

**Technical milestones:**
- Semantic versioning (`v1.0.0`) tag published
- Pre-built binaries for Linux (amd64/arm64), macOS (amd64/arm64), and Windows (amd64)
- All public commands and flags documented in a versioned reference

**Success metrics:**
- Zero open P0 bugs at GA
- Documentation covers 100% of public commands
- Binary size < 15 MB
- Install-to-first-task time < 2 minutes (measured against documented quick-start)

## Files to Modify/Create

### Modify
- `.github/workflows/release.yml` — Trigger Homebrew formula update on release; add additional package manager publish steps

### Create
- `Formula/tssk.rb` — Homebrew formula (or a commit to `homebrew-core` tap repository)
- `.github/workflows/homebrew-update.yml` — Auto-update Homebrew tap on new releases
- Packaging files for other package managers as needed:
  - `AUR/PKGBUILD` — Arch Linux User Repository
  - `snap/snapcraft.yaml` — Snapcraft snap package
  - `tssk.nuspec` — Chocolatey package (Windows)
