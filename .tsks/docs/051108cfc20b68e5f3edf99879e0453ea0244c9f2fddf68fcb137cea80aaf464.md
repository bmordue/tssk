- `internal/tui/` — TUI package

  - `internal/tui/detail.go` — Task detail view component
  - `internal/tui/edit.go` — Task edit view component
  - `internal/tui/styles.go` — Styling constants
- Or alternatively: `web/` — Web frontend directory
