Identify and eliminate duplicated code across the codebase to improve maintainability and reduce bugs.

Tasks:
- Run code duplication detection tools (e.g., cpd, duplo, sonar)
- Identify duplicate code blocks (>3 lines)
- Extract common logic into reusable functions/utilities
- Replace duplicates with function calls
- Update tests to cover refactored code
- Verify no functionality changes

Acceptance Criteria:
- Duplication reduced by at least 50%
- All existing tests pass
- No new bugs introduced

AI-Friendly: Pattern matching for duplicate detection, systematic refactoring
